## Project description and instruction how to run it locally

### Backend

- Go to backend folder and run `npm install` for dependency installation
- Make you create `.env` by copying the content of `.env.example` and then run this project.
- To run the project `npm run dev`
- To run the test case execute `npm run test`

### Frontend

- Go to frontend folder and run `npm install` for dependency installation
- Make you create `.env` by copying the content of `.env.example` and place your `VITE_DYNAMIC_ENVIRONMENT_ID` value before run this project
- To run the project `npm run dev`
